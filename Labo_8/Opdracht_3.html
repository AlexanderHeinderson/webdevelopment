Opdracht 3

  1. De div (waar de css text-align:center; is) zorgt ervoor dat de tekst gecentreerd word

  2. vertical-align: center; bestaat niet.

  3. Het element of de inline-block wordt ge-aligned op 1 lijn

  4. Je zet de line-height op hetzelfde als de height van het element of div

  5. Via "display: table-cell" zorg je ervoor dat het element of div zich gedraagt als <td> element. Dan kun je er "vertical-align: middle" op toepassen.
